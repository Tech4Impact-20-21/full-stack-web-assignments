1.  Apa itu responsive web design?
    : Responsive web design adalah teknik agar web dapat menyesuaikan dengan ukuran device, jadi ketika dibuka di laptop atau smartphone design yang ditampilkan tetap bagus dan tidak berantakan.
2.  Sebutkan apa saja relative units yang dimiliki oleh CSS?
    : %, em, rem, ch, vh, vw, vmin, vmax, ex.
3.  Apa itu breakpoint? Bagaimana cara kita membuat hal tersebut?
    : breakpoint adalah batasan untuk setiap device, untuk mencari titik breakpoint terdapat di developer tools atau tinggal search di browser.
