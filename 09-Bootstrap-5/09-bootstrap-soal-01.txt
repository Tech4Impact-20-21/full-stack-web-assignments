1. Apa itu CSS Framework?
Kerangka kerja CSS yang digunakan untuk desain web agar lebih mudah dan konsisten dengan menggunakan CSS.

2. Apakah bootstrap satu-satunya CSS Framework yang dapat kita gunakan?
Tidak, ada framework lain yang dapat digunakan.

3. Sebutkan beberapa CSS Framework selain Bootstrap
Foundation, Bulma, Semantic UI, Materialize CSS, PureCSS, Skeleton, Tailwind

4. Bagaimana tata cara penggunaan Bootstrap?
Install dengan pilihan cara berikut
    - CDN, dengan menyisipkan link
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-ka7Sk0Gln4gmtz2MlQnikT1wXgYsOg+OMhuP+IlRH9sENBO0LRn5q+8nbTov4+1p" crossorigin="anonymous"></script>

    - Package Manager
    npm : npm install bootstrap
    yarn : yarn add bootstrap
    RubyGems : gem 'bootstrap', '~> 5.1.3'
